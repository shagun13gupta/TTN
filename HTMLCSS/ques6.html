Ques 6: Why do we use meta tags in HTML ?

Meta data is data about data. The meta tag provides metadata about the HTML document.

Metadata will not be displayed on the page, but will be machine parsable.

Meta elements are typically used to specify page description, keywords, author of the document, last modified, and other metadata.The metadata can be used by browsers to display content or reload page search engines (keywords), or other web services.

Meta tag always go inside the head element. It is always passes as name-value pairs.
